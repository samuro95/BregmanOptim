class deepinv.models.UNet(in_channels=1, out_channels=1, residual=True, circular_padding=False, cat=True, bias=True, batch_norm=True, scales=4)[source]ÔÉÅ

Bases: Module

U-Net convolutional denoiser.

This network is a fully convolutional denoiser based on the U-Net architecture. The number of downsample steps can be controlled with the scales parameter. The number of trainable parameters increases with the number of scales.

Warning

When using the bias-free batch norm BFBatchNorm2d via batch_norm="biasfree", NaNs may be encountered during training, causing the whole training procedure to fail.

Parameters:

  • in_channels (int) ‚Äì input image channels

  • out_channels (int) ‚Äì output image channels

  • residual (bool) ‚Äì use a skip-connection between output and output.

  • circular_padding (bool) ‚Äì circular padding for the convolutional layers.

  • cat (bool) ‚Äì use skip-connections between intermediate levels.

  • bias (bool) ‚Äì use learnable biases.

  • batch_norm (bool, str) ‚Äì if False, no batchnorm applied, if True, use torch.nn.BatchNorm2d(), if batch_norm="biasfree", use BFBatchNorm2d from ‚ÄúRobust And Interpretable Blind Image Denoising Via Bias-Free Convolutional Neural Networks‚Äù by Mohan et al..

  • scales (int) ‚Äì Number of downsampling steps used in the U-Net. The options are 2,3,4 and 5. The number of trainable parameters increases with the scale.
